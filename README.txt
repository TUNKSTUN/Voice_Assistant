ALISA

A Personal Desktop Voice Assistant Build using Python Programming and Kivy Scripting, integrated with firebase database to fetch previous year Question papers.



Installation Instruction:

1. Online Command to install packages "pip install playsound==1.2.2 pyaudio gtts "


2. if Crypto package error occurs(Module not found error), go to the root directory of the python installed (say 3.10),then go in 
   lib/site_packages/crypto and rename the folder to "Crypto" (Capital C).

   playsound==1.2.2 should be installed (no latest).


Happy Coding!
